#### /
```
/ 表示根目录
```

#### /root
```
/root 表示系统管理员的用户主目录
```

#### /home
```
/home 表示用户的主目录。/home下存放的都是普通用户，比如创建一个普通用户test 就会在/home文件夹下 创建test文件夹。
```

#### /boot
```
/boot 引导分区
```

#### /bin
```
/bin 是binary二进制的缩写，存放常用命令
```

#### /sbin
```
/sbin 存放系统管理员使用的系统管理程序
```

#### /etc
```
/etc 存放系统配置文件
```

#### /dev
```
/dev 是device缩写，设备目录
```

#### /usr
```
/usr 存放用户的应用程序和文件，类似于windows中的program files
```

#### /lib
```
/lib 是library 系统开机需要的动态连接共享库，类似于windows中的system32存放一些dll文件
```

#### /opt
```
/opt 用于存放一些第三方软件
```

#### /media
```
/media u盘光驱等可移动媒体设备
```

#### /proc
```
/proc 是proccess的缩写
```

#### /srv
```
/srv 是service的缩写
```

#### /var
```
/var 是variable的缩写，可变化的目录，可以用来存放日志
```
