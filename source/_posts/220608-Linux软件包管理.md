## RPM

---

```shell
# rpm 是 RedHat软件包管理工具


# 查询所有安装的rpm软件包
rpm -qa
# 从所有安装包中 过滤出 firefox
rpm -qa | grep firefox
# firfox包的详情信息
rpm -qi firefox


# 卸载软件包
rpm -e 软件包名

# 安装软件包
# -i install，安装 
# -v --verbose，显示详细信息 
# -h --hash，进度条
rpm -ivh 软件包



```

## YUM

---

```shell
# 基于 RPM 包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖的软件包，无须繁琐地一次次下载、安装。


# -y 对所有提问都回答yes
# install 安装 rpm 软件包
# update 更新 rpm 软件包 
# remove 删除指定的 rpm 软件包
# list 显示软件包信息


# 查看安装了哪些包
yum list

# 卸载火狐
yum -y remove firefox

# 安装火狐
yum -y install firefox




```


```shell
# wget 从指定的url下载文件
yum install wget
wget 某个软件包的url




```
